**Flag Quiz App**
A React Native app build with Expo where users can explore country-facts and test their flag knowledge.

**Features**
- Browse countries from a public API
- Search countries by name
- View detailed country information
- Flag quiz with score tracking
- Filter quiz by region and number of questions

**Tech stack**
- React Native
- Expo
- Expo router
- TypeScript
- Rest countries API

**API**
This app uses the REST countries API
- /independent -fetch all countries
- /alpha/{code} -fetch country details

**Installation**
1. Clone the repository

2. Install dependencies
npm install

3. Start the app
npx expo start
